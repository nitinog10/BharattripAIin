### Weather Comfort Index
```python
def calculate_comfort(temp, humidity, wind):
    # Feels-like temperature
    feels_like = temp + (0.5 * humidity/100 * (temp - 14))
    
    # Wind chill factor
    if wind > 5:
        feels_like -= (wind - 5) * 0.5
    
    if feels_like < 20:
        return "Cool"
    elif feels_like < 28:
        return "Pleasant"  # Ideal!
    elif feels_like < 35:
        return "Warm"
    else:
        return "Hot"
```
